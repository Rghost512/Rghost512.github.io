About This Site

This website is being developed for my WEBD-225 course and reflects both my coursework and personal interests. I am using this site to practice working with HTML, CSS, JavaScript, and Bootstrap while building interactive components such as layouts, navigation, and image carousels.

Many of the images on this site feature some of my favorite Fender guitars, which I chose as a way to make the project more personal while still focusing on clean structure, responsive design, and usability. As the course continues, this site will continue to grow and evolve as I add new features and pages.
